## Backend Ingest Verification (Phase 2.1 Exit Gate)

Run this script against a live local backend server to verify the end-to-end flow:

1. Register or login
2. Upsert profile data
3. Run ingest
4. Re-run ingest and verify deterministic behavior

```bash
python scripts/verify_ingest.py \
	--base-url http://127.0.0.1:8000 \
	--gemini-api-key "$GEMINI_API_KEY" \
	--cohere-api-key "$COHERE_API_KEY"
```

The script exits with a non-zero code on any failed assertion, so it can be used in CI.
